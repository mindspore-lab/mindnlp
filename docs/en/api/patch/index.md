# Patch

MindNLP Patch System provides versioned patches for HuggingFace libraries to enable seamless compatibility with MindSpore.

## Components

- [Transformers](transformers/index.md) - Patches for HuggingFace Transformers
- [Diffusers](diffusers/index.md) - Patches for HuggingFace Diffusers
- [Safetensors](safetensors/index.md) - Patches for safetensors
- [Utils](utils/index.md) - Patch utilities
